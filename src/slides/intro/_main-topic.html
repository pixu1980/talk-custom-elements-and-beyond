Web Components is a common term used to describe a set of 3 web platform APIs

Web Components is a family of three browser APIs. In this talk we stay focused on customElements only to keep things simple and practical.

In this talk we will focus only

In particular the customElements v1 API, also known as "customized build-in elements"

Note: unfortunately not supported in Safari...
but a blazing fast Polyfill is available
While customElements have excellent support, the extends pattern (customized built-in elements) requires a polyfill for Safari. This is the main trade-off but the benefits of extending native elements often outweigh this limitation.
